    Get started

    Looks like this is the first time you're using mdash, here are a few steps to get you started:

    1. Open the Bookmark Manager:
      Click the wrench icon on the browser toolbar and select Bookmarks > Bookmark Manager.
    2. Open the folder Other Bookmarks > [Dashboard].
    3. Create some folders inside it, they will be displayed as sections in the dashboard.
      Prefix a folder's name by a '+' to display it on the left or by a '-' to display it on the right.
    4. Now you can move your other bookmarks into these folders or just browse the Web and click the star icon in the address bar before selecting the folder you want to store the bookmark into.

    Note: You can also move the [Dashboard] folder wherever you want (eg. in the Bookmarks Bar folder).

    Features

    • Edit mode: click the top-right wrench button to toggle edit mode on/off. Click a tile to edit its title, URL, or move it to another section.
    • Add bookmark: in edit mode each section shows a + tile — click it to add a new bookmark.
    • Delete bookmark: click a tile in edit mode, then use the Delete link or press Delete inside the edit dialog.
    • Rename section: in edit mode click a section title to rename it inline. Press Enter to save, Esc to cancel.
    • Collapse sections: use the chevron button in each section header to collapse or expand; state is remembered.
    • Delete section: in edit mode use the × button in the section header to remove the folder and all its bookmarks.
    • New section: in edit mode click the big + at the bottom to create a new folder in the left or right column.
    • Drag & drop tiles: in edit mode drag a bookmark tile to another section to move it.
    • Drag & drop sections: in edit mode drag a section header to move the entire section between left and right columns.
    • Undo everything: every action (add, delete, edit, move, rename, create/remove section) shows an Undo notification — click it within 30s to revert.
    • Search: press Option+F (macOS) or Ctrl+F (Windows) to open Spotlight search. Navigate with ↑/↓, press Enter to open. Middle-click or Cmd/Ctrl+Click opens in a background tab.
    • Themes: open the right-side settings panel and switch auto / light / dark.
    • Font size: open settings and choose XXS / XS / S / M / L / XL / XXL / XXXL.
    • Motion: choose full or reduced motion from settings.
    • Refresh icons: click refresh icons to purge favicon cache and reload the page (full rebuild). Alt+Click also purges cache, then rebuilds favicons in place.
    • Help: open settings and click help to toggle this panel. Click outside to close.

    Make sure to also take a look at the online documentation, which features more detailed explanations (with images), and describe every features of the extensions.
